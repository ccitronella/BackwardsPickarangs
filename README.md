## About
**Backwards Pickarangs** is an addon for [Quark](https://modrinth.com/mod/quark) that, as its name suggests, changes the Pickarang to rotate backwards as it is thrown (Specifically, counter-clockwise). However, this is achieved by simply multiplying its rotation by a number, which can be adjusted in its config menu - it can even stop it from rotating entirely, if you want that for some reason.
